Gemini API Key

Some workshop activities use Google's Gemini AI. Enter your key below and it will be passed automatically to interactive pages.

Don't have a key? Get one free
  1. Visit Google AI Studio and sign in with your Google account.
  2. Click Create API Key and select a Google Cloud project (or let it create one).
  3. Copy the generated key and paste it into the input field above.
How is my key kept secure?
  • Tab-scoped storage. Your key is held in sessionStorage, which is scoped to this browser tab. It is automatically cleared when you close the tab and is never written to disk or shared across tabs.
  • Static site. This page is served as a static file from GitHub Pages. There is no server that stores or has access to your key.
  • HTTPS in transit. API calls to Google use HTTPS (TLS), so your key and prompts are encrypted between your browser and Google's servers.
  • URL stripped. If the key is passed via ?key=, it is read once and immediately removed from the address bar so it does not appear in browser history or bookmarks.
  • Your key, your account. Each participant uses their own free-tier Gemini key. No shared credentials are stored anywhere.

Note: As with any client-side application, your key is visible in browser developer tools while the page is open. For a workshop setting with free-tier educational keys this is acceptable. Do not use a key attached to a paid production project.

Facilitators: Pre-fill by sharing a link with ?key=YOUR_KEY appended.
